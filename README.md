# Vanguard Run

Локальный кооперативный паркур-платформер для двух игроков на одном экране:
Игрок 1 — клавиатура, Игрок 2 — геймпад. После каждого забега команда
тратит общий банк монет на прокачки через шесть блоков "?" в безопасном хабе.

## Управление

- **Игрок 1 (клавиатура):** A/D или ←/→ — движение, Пробел/W/↑ — прыжок
- **Игрок 2 (геймпад, устройство 0):** стик влево/вправо или D-pad — движение, A/X — прыжок

## Игровой цикл

1. Пробегите уровень вдвоём от старта в хабе до финиша.
2. За успешный забег команда получает монеты в общий банк.
3. Вернитесь в хаб и подпрыгните под нужный блок "?" снизу, чтобы купить прокачку:
   - **Награда** — увеличивает монеты за забег
   - **Множитель награды** — множит награду за забег
   - **Клон** — добавляет призрака, который в цикле повторяет последний записанный забег и приносит пассивный доход
   - **Награда клона** — увеличивает доход с каждого клона (дороже, чем игровая награда)
   - **Множитель награды клона** — множит доход с клонов
   - **Прокачка уровня** — генерирует новый, более длинный и сложный уровень;
     сбрасывает ТОЛЬКО клонов и их прокачки (количество клонов, награда клона,
     множитель награды клона), а прокачки игрока и банк сохраняются.

## Технологии

- Godot 4.3, GDScript
- Автосборка Windows-инсталлятора через GitHub Actions (`.github/workflows/build-windows.yml`):
  экспортирует релизный `.exe` и упаковывает его в самостоятельный установщик
  через Inno Setup. Готовый инсталлятор публикуется как GitHub Release при
  каждом push в `main`.

## Локальная разработка

Открыть папку `godot-game` в редакторе Godot 4.3+ — весь уровень, экономика
и хаб строятся программно в `scripts/Main.gd`, `.tscn`-сцена содержит только
корневой узел.
